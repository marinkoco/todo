•	Add Task: Prompt the user to enter a task and append it to the text file.
•	View Tasks: Read the contents of the text file and display the tasks to the user.
•	Delete Task: Prompt the user to enter the task number they want to delete, read the tasks from the file, remove the specified task, and rewrite the remaining tasks to the file.
•	Quit: End the program.
The program will prompt you with the menu options, and you can choose the corresponding numbers to perform actions on the to-do list.
